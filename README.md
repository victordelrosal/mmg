# MMG - Professional CV

A modern, interactive CV/resume website for Mario Alberto Montejo Garay, MSc.

## Description

A premium dark-themed professional CV website featuring a bilingual Systems Analyst and Technical Support Engineer with 20+ years of experience. Includes interactive flip-card animations and a sophisticated glass-morphism design.

## Features

- Interactive flip-card animations
- Dark premium design with blue accents
- Glass-morphism effects
- Responsive layout
- Social media preview metadata (Open Graph, Twitter)
- Professional profile photo integration
- Shimmer and fade animations

## Tech Stack

- HTML5
- CSS3 (Custom Properties, 3D Transforms, Animations)
- Google Fonts (Inter)
- Responsive design

## Usage

Open `index.html` in a web browser to view the CV. Click on card elements to see flip animations.
